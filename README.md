# Cahaya & Kal Wedding Invitation

A luxury cinematic wedding invitation built with Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion, GSAP, and lucide-react.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included

- App Router project structure
- Responsive editorial sections
- Sticky glass navigation
- GSAP hero parallax
- Framer Motion scroll reveals
- Countdown timer
- Masonry gallery with lightbox
- Accessible RSVP form with success state
- SEO metadata and Open Graph content
